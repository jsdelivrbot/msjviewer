Mobile MSJ

The Unofficial Android MSJ app

This is an unofficial Android app to help you use MSJ from your Android phone. It is not affiliated with St John in any way, other than being created by a St John member.

It is a hobby project, and I take no responsibility if it doesn't work, kills a kitten, or catches fire.

How does it work?

This app works like your web browser - it literally visits the official MSJ website and displays the contents to you in the app. However, before displaying them, the app changes the "styles" to make it fit better on a smaller mobile phone screen.

This is why the screen may flicker a little bit after displaying a page. It is first loading the original, and then it is restyling it to display nicely on a mobile device.

(View technical details)

The app uses an Android WebView to render the MSJ website. Once a page is loaded, a javascript: URL is loaded, which:

  • Creates one or more link DOM nodes.
  • Sets the href attribute to a CSS file (hosted on GitHub, but proxied through https://cdn.rawgit.com).
  • Attaches the link node to the head of the page.

This has the effect of applying the CSS in the linked file to the page being viewed.

The app intentionally does not:

  • Proxy traffic from MSJ.
  • Modify (or even read) the HTML of the page once loaded.
  • Manually save your username and password (the "remember password" feature is provided by the Android WebView, not by this app).

Why does it exist?

It was built because I was frustrated at the non-mobile-friendly MSJ website. Most times that the website is useful is at divisional meetings and at events, when you are around other volunteers and deciding together what events to attend. It is at these times that you most need a mobile friendly version of the website. I released the app it in the hope that others will also find it useful.

Privacy

This app respects your privacy. It doesn't send information anywhere other than the official MSJ website (https://ssl.stjohnvic.com.au). If you are technically minded, and would like to verify this, the code is free and open source (https://github.com/pserwylo/msjviewer).
